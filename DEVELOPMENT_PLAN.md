# 阳光小镇物语 1.0 开发计划与实施进度

本文档是项目的主开发计划书，也是实施进度清单。目标是把当前 Three.js 小镇经营原型推进到完整 1.0 独立游戏：有稳定开发环境、可保存的长期经营、章节目标、建筑成长、城市事件、清晰反馈、统一视觉风格和可发布质量。

## 1. 当前版本状态

- 当前阶段：P2 收尾验证完成。
- 当前目标版本：`0.3 章节化试玩版`。
- 当前定位：桌面浏览器本地运行的 3D 休闲小镇建设游戏。
- 技术栈：Python 静态服务、原生 JavaScript、Three.js、Playwright。
- 主要玩法：18 x 18 网格、道路规划、住宅/商业/工业/服务建筑、居民通勤、道路拥堵、周结算、章节目标、存档、建筑升级。

## 2. 核心产品目标

一款明亮、温暖、轻策略的 3D 小镇建设游戏。玩家通过道路、服务、就业和环境规划，让居民在晴日港稳定生活，并逐步把小镇发展成有阶段目标、有成长反馈、有独立视觉气质的完整作品。

1. 新玩家 3 分钟内能开始建设。
2. 前 15 分钟能理解道路、住宅、水电、就业和通勤的关系。
3. 30 到 60 分钟内能完成第一批章节目标并产生继续扩建动力。
4. 完整 1.0 支持 2 到 4 小时轻量经营。
5. 每个关键系统都有 UI 解释和自动化测试覆盖。
